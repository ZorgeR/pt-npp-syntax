### Подсветка синтаксиса PT MaxPatrol SIEM в Notepad++

#### Поддерживается:

1) Fold / Unfold
2) Подсветка переменных
3) Подсветка ключей
4) Подсветка функций
5) Подсветка логических операторов

#### Установка:
1) Импортировать файл *.xml в npp
2) Выбрать подсветку синтаксиса PT-Rules-Correlation
